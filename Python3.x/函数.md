## 函数 ##

###### 函数式编程：

```
def test(x):
    "The function definitions"
    print("函数式编程")
    return 0

```
 
###### 面向过程编程：

```
def test(x):
    "The function definitions"
    print("面向过程编程")
```

###### python中函数定义方法：

```
def test(x):
    "The function definitions"
    x+=1
    return x

y = test(2)
print("from test return is %s" %y)

# def:定义函数的关键字，x是形参
test：函数名
()：括号内可定义形参
“ ” ：文档描述
x+=1 代码块或程序处理逻辑
return：定义返回值
y：是调用函数，2是实参

```
函数(函数调用)其中之一的好处就是减少代码重复率

```
def test():
    print("this is a test")
    return 0
    print("the end")

test()

a = test()
print(a)

```
*打印输出*
```
D:\python34\python.exe E:/PycharmProjects/untitled/study/day3/函数.py
this is a test
0
Process finished with exit code 0

【说明】return 0之后的语句就不会再执行了
如果想得到返回值，可以直接把函数赋值给一个变量(a)，a得到的就是返回值
```
