## 面向对象-类

##### 类的一个简单实例：
```ruby
class dog:

    def __init__(self,name):
        self.name = name

    def bulk(self):
        print("%s:wang wang wang!" %self.name)

dog1 = dog("Dog1")
dog2 = dog("Dog2")
dog3 = dog("Dog3")

dog1.bulk()
dog2.bulk()
dog3.bulk()

```
*`输出:`*
```ruby
Dog1:wang wang wang!
Dog2:wang wang wang!
Dog3:wang wang wang!
```
